Summary:
Retrieves users' data from database (MS SQL Server) based on a given search criteria
  • LINQ to interract with database
  • webview2, AngularJS, Bootrap3, HTML5 for UI design

Features:
  • Translates text from one language to another using Microsoft Translate API (Default being from English to French)
  • Possibility to read translated file as an audio through the use of Microsoft Speech synthesis (either a male or a female voice could be selected. The default being female voice)
  • Possibility to import a text/pdf file for translation
  • Possibility to export translated text as an audio file
  • DevExpress tools for UI design
